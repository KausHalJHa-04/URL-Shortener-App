## 📖 About The Project

This project is a **Full Stack URL Shortener Application** developed using the **MERN Stack (MongoDB, Express.js, React.js, and Node.js)**.

It allows users to convert long URLs into short, shareable links within seconds. Every shortened URL is paired with a dynamically generated **QR Code**, making it easy to access links across different devices.

The application focuses on speed, simplicity, and a clean user experience while demonstrating complete frontend and backend integration.

## 🚀 Installation

### Clone the Repository

```bash
git clone https://github.com/KausHalJHa-04/URL-Shortener-App.git
```

Move into the project folder

```bash
cd URL-Shortener-App
```

Install dependencies

```bash
npm install
```

Start the frontend

```bash
npm run dev
```

Start the backend

```bash
npm run server
```

---

## ⚙️ Environment Variables

Create a `.env` file inside the server folder.

```env
PORT=5000

MONGODB_URI=your_mongodb_connection

BASE_URL=http://localhost:5000
```

---

## 🎯 How It Works

1. Enter a long URL.
2. Click **Shorten It**.
3. Backend creates a unique short URL.
4. The URL is stored in MongoDB.
5. A QR Code is generated automatically.
6. Copy the shortened URL or download the QR Code.
